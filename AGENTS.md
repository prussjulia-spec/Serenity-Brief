# Правила работы с проектом

- Перед началом любой задачи сначала выполнить `git pull`.
- По каждой задаче писать реальные тесты и отдавать задачу только после успешного прохождения тестов по всем условиям задачи и контексту.
- Когда задача принята и результат закоммичен, временные тесты удалить.
- Ничего не придумывать от себя. Действовать строго в рамках оригинала; если что-то непонятно, уточнять.
- Коммиты делать на русском.
